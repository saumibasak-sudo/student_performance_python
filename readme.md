# Student Performance Analysis

## Project Overview
This project analyzes student exam performance based on study hours,
attendance, and gender using Python.

## Folder Structure
- data/: Dataset files
- visualizations/: Generated charts
- report/: Written analysis report

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the analysis:
   python main.py
